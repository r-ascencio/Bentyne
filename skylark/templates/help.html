Themes


Creating a theme for your CMS is even easier now. You just have to define where is going to be loaded your content, and chose your theme.

For a basic theme you need define these files and dirs:

default.html (required)

This is the template that we're going to use if you haven't definded one yet.

blog_index.html (required)

This is the template that we use in the blog_index

blog_detail.html (required)

This is the template that we're going to use for each blog on the post.

404.html

Page not Found.

500.html

Error on the server.

<"theme_name">/static/

In the directory is where you have to save the media file that you use on your theme.
